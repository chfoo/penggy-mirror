3. Usage

3.1 Configuration

Pour configurer Pengaol cliquer sur config ou editez le fichier /etc/PengAol.conf

Dans la section [CLoader] vous pouvez selectionner votre driver d´entree et de sortie

Les driver d´entrees disponibles sont :

CModemDriver pour :

-Modem Rtc (device=/dev/modem ou /dev/ttySx avec pour x de 0 a 4, generalement /dev/modem)

-Modem Numeris (device=/dev/ttyIx avec pour x de 0 a 4)

Pour utiliser un modem numeris , dans la section [CModemDriver] modifier comme ceci :

- Device= Votre peripherique Numeris (generalement ttyI0)

- Initstring1=AT&F

- Initstring2=ATExxxx avec pour xxxx les 4 dernier numero de votre ligne telephonique (ex pour 03.12.34.56.78 tapez 5678)

CCableDriver pour:

Toute connection TCP/IP.

Verifier dans la section [CCableDriver] que device= pointe bien sur votre interface reseaux (generalement eth0)

Driver de sortie disponibles:

CTunTapDriver : Utilise Ethertap (/dev/tap0) ou Tun/tap (/dev/net/tun)

CPppDriver : Utilise l´interface /dev/ppp .

3.2 Usage general

Entrez votre nom d´utilisateur et votre mot de passe, cliquez sur Connect, C tous !!!

Cepandant si vous utilisez une version console :

En premier enregistrez un utilisateur avec : pengaol -AddUser [login] [Pass]

Maintenant vous pouvez vous connecter avec : pengaol -Connect [login]

Pour arretez pengaol en mode console tapez ctrl+c ou envoyer un signal SIGINT
