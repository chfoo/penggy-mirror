2. Installation

2.1 Comment obtenir Pengaol

Pour obtenir la version la plus recente, visiter svp le site www.pengaol.org

2.2 Configuration requise

Au minimum vous devez disposez d´un noyau 2.2.x avec le modules ethertap

Configuration requise :

Un noyau 2.2.x avec le module ethertap pour utilise CTunTapDriver

Un noyau 2.4.x avec le module tun pour utiliser CTunTapDriver

Pppd 2.4.x avec comme modules ppp_genric,ppp_synctty,ppp_asynch modules pour utiliser CPppDriver

2.3 Compilation et installation

Pour compiler et installer PengAol tapez ceci a la base de votre distribution : 

% ./recompile GUI (Pour creer la version graphique)
% ./recompile NOGUI (Pour creer la version console)

Pengaol utilise pour se configurer 

autoconf
Vous ne devez pas avoir de probleme pour le compiler. Si vous rencontrez des probleme soumettez vos Erreurs a l´auteur : Guth stephane (birdy57)
