1. Introduction

Pengaol est une passerelle IP linux pour AOL.

Il supporte pour l´instant les connections via modem (RTC et NUMERIS) et via TVP/IP (Numericable)

Actuellement, seul le protocol proprietaire P3 est emuler cepandant le noyau ELV6 est en cour de devellopement (Emulation de AOL(c) 6)

Ce programme est un reverse-ingenering du client original, cepandant aucunes lignes de code n´as ete dessassemblees !!

Pour comprendre le protocol P3, j´ai simplement sniffe la communication avec le modem.

En raison meme de la licence de pengaol, et du faite que les sources sont fournit, vous ete garanti que pengaol n´envoie aucune informations se trouvant sur votre ordinateur ;-)

Pengaol est tres compact et rapide, je n´y inclue aucun element superflue car je considere que c a l´utilisateur de choisir ses outils prefere (AIM, CHAT etc...)

Have Fun, Birdy57
