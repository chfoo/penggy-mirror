2. Installation

2.1 How to obtain Pengaol

To obtain the newest version from pengaol, please visit www.pengaol.org

2.2 Requirements

Mininum require is a 2.2.x kernel with ethertap modules.

You require :

A 2.2.x kernel yith ethertap as module to use the CTunTapDriver

A 2.4.x kernel with tun/tap as module to use the CTunTapDriver

Pppd 2.4.x with ppp_genric,ppp_synctty,ppp_asynch modules to use the CPppDriver

2.3 Compilation and installation

In order to compile and install Pengaol on your system, type the following in the base directory of the Pengaol distribution: 

% ./recompile GUI (To compile a graphik version)
% ./recompile NOGUI (To compile a console version)

Since Pengaol uses 

autoconf
you should have not trouble compiling it. Should you run into problems please report them to the the author at Guth stephane (birdy57)
