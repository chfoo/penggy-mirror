1. Introduction

Pengaol is a Linux IP tunneling client for AOL.

It support to time : modem connection (RTC/ISDN) and TCP/IP connection.

Only the P3 protocol are now availlable, but i start the ELV6 kernel (Emulation from the AOL(c) 6 protocol)

This sofware is based on a Reverse-ingenering from the original client, but i don´t have disassembled anything !

To understand the P3 protocol, i have only sniff my modem.

Because this sofware are GNU/GPL, you can sure that pengaol don´t send anything from your computer ;-),

PengAol is small and fast, i don´t want to include anything. I thing , it is to the user to choose is prefered tools (AIM,Chat etc..)
