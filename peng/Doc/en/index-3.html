3. Usage

3.1 Configuration

To use PengAol first click to Config or edit the /etc/PengAol.conf file.

In section [CLoader] you can select the input and ouput driver to use.

Availlable input driver are:

CModemDriver for :

-Rtc modem (device=/dev/modem or /dev/ttySx with x from 0 to 4)

-Isdn Modem (device=/dev/ttyIx with x from 0 to 4)

For use a Isdn modem, please change in section [CModemDriver]

- Device= your isdn device (normally ttyI0)

- Initstring1=AT&F

- Initstring2=ATExxxx with xxx = the last 4 numbers from you phone number.

CCableDriver for:

A Tcp/Ip connection.

In section [CCableDriver] change device= to you connecting interface (normally eth0)

Availlable ouput driver are:

CTunTapDriver : To use Ethertap (/dev/tap0) or Tun/tap modules (/dev/net/tun)

CPppDriver : To use the /dev/ppp interface.

3.2 General Usage

Enter you Login, Password and click Connect , That´s all !

But if you don´t want the graphik interface :

First add a user with : pengaol -AddUser [login] [Pass]

than you can connect you with : pengaol -Connect [login]

To stop pengaol in console mode, tape ctrl+c or send a SIGINT signal.
